Установка новой версии ядра 5.15 в РедОС 7.3

Перед установкой новой версии ядра обновите систему, выполнив в терминале команду:

dnf update -y

После окончания процесса обновления пакетов операционной системы установите пакет для подключения раздела репозитория kernels:

dnf install redos-kernels-release -y

Затем выполните команду для обновления кэша пакетов:

dnf makecache

Выполните команду для обновления ядра и ядрозависимых пакетов:

dnf update -y

После этого перезагрузите ПК:

reboot

Удаление старых ядер

Для того чтобы посмотреть установленные в системе пакеты ядра, можно воспользоваться консольной командой:

rpm -qa kernel-*

Удаляем ядра

dnf remove kernel-lt-5.10* -y
